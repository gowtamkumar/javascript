🧠 2. Step-by-step Intuition (How to Think)

When solving array problems, always think in this order:

Step 1: Understand the problem
What is input?
What is expected output?
Step 2: Identify pattern

Most array problems fall into these patterns:

Linear scan (loop)
Two pointers
Sliding window
Prefix sum
Step 3: Ask key questions
Do I need all elements or just part?
Is order important?
Is array sorted?
Can I use extra memory (like object/map)?
Step 4: Optimize

Start with brute force → then improve
